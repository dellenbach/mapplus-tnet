Wanderwege

Auf MAP+ haben wir unter Wanderungen Wandervorschläge von verschiedenen Quellen integriert:

  • Wandersite: Ein Auszug aus den Routen von Wandersite (mit Link zur Beschreibung)
  • Alternatives Wandern:  Ein Auszug aus den Routen von Alternatives Wandern  (mit Link zur Beschreibung)
  • MAP+ (siehe unten)

Wanderwege MAP+

Die Idee ist hier, Wandervorschläge der MAP+ User zu sammeln. User können auf zwei Arten Wanderwege erfassen:

  • via GPS-Tracks: GPS Datei hochladen: gpx+
  • manuell: Wanderung manuell erfassen: +

Wie nehme ich Wanderwege auf?

SmartPhones oder GPS Geräte eignen sich am besten. Folgende Tipps zu SmartPhones:

  • Android: Die beste Software ist Orux Maps
  • iPhone: Hier sieht Galileo vielversprechend aus
